# User Manual

## Start the app
1. Open the project folder in VS Code
2. Activate the virtual environment
3. Run `python app.py`
4. Open the local Gradio URL in the browser

## Use text input
- Type a question in the text box
- Select the answer language
- Click **Ask the chatbot**

## Use audio input
- Record audio or upload an audio file
- Select the spoken question language
- Click **Ask the chatbot**

## Clear the chat
- Click **Clear chat** to reset the conversation

## Best demo tips
- Keep one typed question ready
- Show English first
- Then switch to another language
- Then play the answer audio
