# Models

This folder stores lightweight processed artifacts for the chatbot. For this dry run, the chatbot uses a simple keyword-based knowledge base, so no trained model file is required.


- This project is a proof-of-concept educational chatbot built mainly with a rule-based approach instead of a fully trained machine learning model.

- At this stage, there is no saved trained model file inside this folder because the chatbot responses are generated from a structured knowledge base and topic-matching logic implemented in the source code.

- This `models/` folder is still included to follow a professional project structure and to support future improvements. In the next version of the project, this folder can store trained NLP or classification models for more advanced chatbot capabilities.
